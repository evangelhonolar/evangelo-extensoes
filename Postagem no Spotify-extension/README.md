Extensão Chrome gerada automaticamente pelo **Chrome Interaction Recorder**.

## Descrição

Esta extensão reproduz automaticamente 37 ações gravadas previamente.

## Instalação

1. Baixe e extraia este arquivo ZIP
2. Abra o Chrome e navegue para `chrome://extensions/`
3. Ative o "Modo do desenvolvedor" (canto superior direito)
4. Clique em "Carregar sem compactação"
5. Selecione a pasta extraída

## Como Usar

1. Navegue até a página onde deseja executar a automação
2. Clique no ícone da extensão
3. Clique em "▶️ Iniciar Automação"
4. A extensão executará todas as ações gravadas automaticamente
